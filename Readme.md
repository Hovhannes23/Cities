**Консольная игра в городки**

Программа при запуске заполняет список названиями городов (названия берутся из внешнего файла) и предлагает пользователю написать свой город.
Затем удаляет из списка город пользователя (чтобы повторно его не использовать) и ищет все города, которые начинаются на последнюю букву города пользователя.
Все полученные города кладет в новый список, из которого рандомно достает один и удаляет его из общего списка. 
Предлагает пользователю ввести свой город и проверяет, чтобы первая буква совпала с последней. И т.д.

Программа пока работает на примитивном уровне.<br>

**В планах:**
- увеличить список городов<br>
- если название города оканчивается на "ь", учитывать предпоследную букву. Если - на "й", считать её как "и".<br>
- реализовать общение пользователя с приложением на основе Http-запросов и запускать на локальном сервере.<br>
- при озвучивании города парсить его фотографию через яндекс-картинки и выводить это фото на странице.<br>
  Правда пока нет понимания, как это можно реализовать, и критическими ли будут задержки.<br>
